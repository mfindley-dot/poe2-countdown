# 🔮 GLG PoE2 Stash Scanner Browser Extension

A lightweight, zero-terminal, zero-command Chromium browser extension that allows any guild officer to capture their PoE2 screen visually and sync their actual in-game currency stash tab to the countdown website database in single click!

---

## 🚀 Easy 30-Second Setup Guide

To load the extension into Google Chrome (or any Chromium browser like Edge, Opera, or Brave):

1.  **Open Chrome Extensions**:
    Open a new tab in Chrome, type **`chrome://extensions/`** in the URL bar, and press Enter.
2.  **Enable Developer Mode**:
    Toggle the **"Developer Mode"** switch in the **top-right corner** of the page to **ON**.
3.  **Load the Extension**:
    *   Click the **"Load unpacked"** button in the top-left corner.
    *   Select the **`chrome_extension`** folder from this project directory.
    *   *That's it!* The puzzle-piece icon in your Chrome bar will now show the **🔮 GLG Stash Scanner** extension!

---

## ⚡ How to Scan and Sync Live Stash counts

1.  Open Path of Exile 2 in **Borderless Windowed Mode** with your **Currency Stash Tab** open on screen.
2.  Click the **🔮 GLG Stash Scanner** icon in your Chrome extensions bar.
3.  Both the **Gemini API Key** and **Dreamlo Private Key** are pre-filled with your guild's shared keys by default! Less tech-savvy guildmates can run it instantly with zero configuration! *(Power users can paste their own keys to override them).*
4.  Click **🔮 CAPTURE & SYNC STASH**.
5.  A standard browser sharing window will prompt: **"Choose what to share"**.
    -   Select the **"Window"** tab and choose **Path of Exile 2**, or select **"Entire Screen"**.
    -   Click **Share**.
6.  The extension will instantly grab the frame, crop, run the Gemini Multimodal Vision analysis, and sync the data online in 5 seconds!
7.  Check your guild's website **`vault.html`** page to watch your real currency counts and net worth render live!
